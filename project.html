Visualizing Bicycle and Exercise Segments with JavaScript

Created by: Connor Gooch, Ethan John, Cameron Hurst, Sierra Kostic, and Winston Waldrip

Group Project for WebGIS at Texas A&M University

General Use and Function

In order to view Strava segments over a specific area, enter the desired latitude, longitude, and zoom level and press the “Update Map” button.
Once pressed, the map and table will be updated to display the top 10 segments through Strava API, which have been visualized to display
a range of difficulty depending on the distance of the segment through a color gradient of green being of short distance and red being of long distance.







This project was made possible thanks to the Strava API, Leaflet, and Esri. This project uses Strava API to fetch and display Strava segments based on their geographic location and activity type, and then classify them from green to red based on the segment's distance. This is a tool that we hope people can use to better visualize and understand workout segments, as well as a general difficulty based on the color gradient. For more information on Strava and Strava API, please visit Strava API Documentation.
